## Product Stack

### 1. x402-secure
- **What:** Open-source SDK + proxy layer for agent-to-agent payments
- **Adds:** Verified identity, intent checks, agent-native risk controls to every x402 payment
- **Integration:** "Integrates in minutes"
- **Dashboard:** Real-time ranking, security scores, anomaly detection, official x402 server leaderboard (supported by Coinbase x402 Bazaar)
- **Target:** Developers / platforms building on x402

### 2. ClawCredit
- **What:** First "agent-native credit" — underwritten by t54's risk engine
- **Key Features:**
  - **Keyless:** Agents don't manage private keys or pre-fund gas. Settlement via custody.
  - **Credit Building:** Agents build credit profiles through usage/repayment, unlocking higher limits.
  - **Auditability:** Every spend linked to an agent reasoning trace + code snapshot.
- **Target:** Agent operators needing compute / service payments

### 3. XRPL x402 Facilitator
- **What:** AI agents pay for services using $XRP and $RLUSD
- **Ecosystems:** XRPL, Solana, Base, Virtuals

## What They Do (Skip / Differentiate Against)

| # | Item | Why We Skip / Differentiate |
|---|------|------------------------------|
| 1 | **Custodial / keyless approach** | They handle custody. We can differentiate on **self-custody** + vault-controlled keys. Power users prefer this. |
| 2 | **XRPL-first / heavy institutional rails** | They're XRPL-centric + TradFi-backed. We stay **chain-agnostic** (Base + Solana where our skills live) and DeFi-native. |
| 3 | **Pure B2B / enterprise-only** | They target businesses. We target **individual power users, small agent builders, hackathon devs** — the long tail. |
| 4 | **Closed proprietary risk engine** | Trustline is black-box. We can make REP **transparent / community-governed** — open scores, open weights. |
| 5 | **No user upside / yield component** | They are pure trust/security. We bundle **yield farming + personal finance tracking** — users earn while agents work. |

## Content & Messaging Takeaways

1. **"Unverified and unaccountable"** is their anchor phrase. Our counter: **"Verified by reputation, backed by yield."**
2. **Stats land hard.** They cite YouGov 42% + Keyfactor 86%. We need our own data points (even small surveys or on-chain metrics).

4. **Bounties signal maturity.** $10K isn't huge, but it gets attention. We should plan one before mainnet.
5. **Institutional quotes land.** Franklin Templeton + Ripple exec quotes give them gravity. We need validator/backer quotes as we grow.
