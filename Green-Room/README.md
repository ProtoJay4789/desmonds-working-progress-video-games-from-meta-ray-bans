# 🎬 Green Room

This is where we think, plan, and log work in progress.

## Structure

- **ideas.md** — Checkbox list of ideas to explore
- **Ideas/** — Detailed idea documents
- **build-logs/** — Active work logs. Log your process while building.
- **WORKFLOW-ACTIVE.md** — Current operating procedure

## How to Use

When working on something:
1. Create a build log in `build-logs/` with the date and topic
2. Write down your process as you go — decisions, blockers, discoveries
3. Move to `Archive/` when complete

Ideas that aren't tied to active work go in `ideas.md` or `Ideas/`.
