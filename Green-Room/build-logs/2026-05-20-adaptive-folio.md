# AdaptiveFolio — Agora Agents Hackathon

## Concept
AI-powered adaptive portfolio manager that detects market regimes and auto-rebalances across DeFi/TradFi assets, settled on Arc with USDC.

## Target RFB
RFB 04 — Adaptive Portfolio Manager

## Key Features
- Regime detection (risk-on / risk-off / sideways)
- Goal-based portfolio management
- Cross-chain rebalancing via CCTP
- USDC-settled on Arc (~$0.01 fees)
- Tax-loss harvesting awareness

## Tech Stack
- Arc blockchain (Circle L1)
- Circle USDC / CCTP
- Python agent core
- FastAPI dashboard
