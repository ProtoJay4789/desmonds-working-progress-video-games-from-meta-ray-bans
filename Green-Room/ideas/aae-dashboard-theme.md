# AAE Dashboard Theme

## Overview
Custom Hermes dashboard theme with AAE (Agent Arena Economy) branding. Would make the dashboard feel like part of the GenTech ecosystem instead of generic Hermes.

## Requirements
- Custom theme/skin for Hermes dashboard
- AAE branding (logo, colors, typography)
- Consistent with GenTech Labs visual identity
- Works with new v0.16.0 font customization

## Notes
- v0.16.0 now supports independent font + theme customization
- Theme picker available in dashboard settings
- May need to create a custom CSS theme file
- Check `hermes config set display.skin` for available themes
