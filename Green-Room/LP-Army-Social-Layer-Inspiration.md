# LP Army Pattern — Social Layer Inspiration

## Source
LP Army (@met_lparmy) — Meteora community with 17K followers
- DLMM Playground V2 — risk-free strategy testing
- Community-driven liquidity provision
- Strategy sharing and optimization

## The Pattern

**LP Army Formula:**
1. **Playground** — risk-free testing environment
2. **Community** — shared strategies, leaderboards
3. **Social Proof** — see what others are doing
4. **Engagement** — compete, learn, share

**This is not just DeFi tooling — it's social infrastructure around financial strategies.**

## AAE Portal — Social Layer

### Strategy Playgrounds
- Risk-free simulators for agent strategies
- Humans watch agents test strategies in real-time
- Educational layer — learn how agent strategies work before investing

### Community Features
- Strategy sharing — agents post their strategies, humans discuss
- Leaderboards — best performing agent strategies, top humans backing them
- Comments/discussion — humans can ask agents about their approach
- Social proof — show which strategies are trending

### Investment Flow
```
1. Human discovers agent strategy on leaderboard
2. Reads agent's track record + community feedback
3. Allocates funds (RAMS mandate for compliance)
4. Agent manages portfolio autonomously
5. Human tracks performance, earns yield
6. Shares results, builds social proof
```

## Agent Arena — Game Mode

### Liquidity Provision Mode
- Agents compete to provide best LP positions
- Real-time simulation on Meteora DLMM
- Score based on: yield, impermanent loss, capital efficiency

### Strategy Sharing
- Agents learn from each other's LP tactics
- "Strategy marketplace" — agents can sell/share strategies
- Human players can adopt agent strategies in-game

### Social Features
- Watch live LP positions being managed
- Chat with agents about their strategy
- Vote on which strategies should be "featured"

## Key Insight

LP Army shows that **financial tools become social platforms when you add:**
1. Risk-free testing (playground)
2. Community (strategy sharing)
3. Competition (leaderboards)
4. Social proof (what others are doing)

We can replicate this for:
- Agent-managed funds (AAE Portal)
- DeFi game modes (Agent Arena)
- Strategy education (both products)
