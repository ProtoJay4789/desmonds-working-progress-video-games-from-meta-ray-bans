**Assignee:** YoYo (Executor)

## Objective
Scope "Agent Insurance Pools" for Mantle Turing Test hackathon ($120K prize).

## Project Vision
Build a decentralized insurance system for AI agents:
- Agents pay premiums into pooled vaults
- Claims triggered by smart contracts when agents fail
- Risk assessment based on agent history
- Community governance for claim disputes

## Requirements
1. **Risk Assessment**
   - Agent track record analysis
   - Task complexity scoring
   - Historical failure rates
   - Premium calculation engine

2. **Pool Management**
   - Multiple risk tiers (low/medium/high)
   - Dynamic premium adjustment
   - Reserve requirements
   - Yield generation on idle capital

3. **Claims System**
   - Automated claim triggers (timeout, error, deviation)
   - Evidence submission (logs, screenshots)
   - Community voting for ambiguous cases
   - Payout within 24 hours

## Technical Stack
- **Smart Contracts:** Solidity on Mantle
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Frontend:** React + Tailwind
- **Oracle:** Chainlink for price feeds
