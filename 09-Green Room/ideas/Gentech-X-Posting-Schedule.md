# Gentech X Posting Schedule

> Flexible framework — Jordan updates shift times each Sunday in HQ thread.
> Slots map to availability, not fixed clock times.

## Daily Cadence (pick based on that day's schedule)

### Slot 1 — Morning Warm-Up (10-15 min)
- Reply to threads, quote tweets, share a stat
- Low effort, keeps algorithm warm

### Slot 2 — Lunch Window (15-20 min)
- Medium post: chart + take, short hot take, thread reply
- Pre-drafted by cron, just review + post

### Slot 3 — Post-Shift / Evening (30-45 min)
- Deep content: full thread, research post, vault draft polish
- Pre-drafted by cron, review + edit + post
