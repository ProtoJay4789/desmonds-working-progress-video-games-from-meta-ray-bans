# Singularity Layer (x402_Layer) — Research Notes
*Added: 2026-06-13*

## What It Is
"The Internet's unified commerce layer for Homo-Agentic economy."
Infrastructure for AI agents to create, run, and scale digital businesses.

## Core Products

### 1. Singularity Studio
No-code toolkit for creating x402-powered API endpoints.
- Launch endpoints in under a minute
- No code required
- Analytics built-in
- Digital goods support

### 2. x402 Payments
Pay-per-request HTTP payments on Base and Solana.
- 0% transaction fees
- Instant settlement
- Multi-chain (Base + Solana)
- Webhooks for agentic access
- Credit packs and subscriptions

### 3. Singularity Marketplace
Discovery layer for agentic services.
- List your API endpoints
- Buy/sell agent capabilities
- Agent discovery and trust

### 4. Singularity Compute (SGL Grid)
Distributed compute network.
- Run grid nodes (v1.2.0 just dropped)
- Token streaming at node level
- Confidential computing support
- Earn SGL by providing compute

### 5. Staking Engine
- Stake SGL tokens
- Earn USDC + SGL rewards
- Multiple staker types
- Slashing and safety mechanisms

### 6. ERC-8004 Agent Identity
On-chain identity standard for AI agents.
- Soulbound tokens for discovery
- Reputation registry
- Trust framework

### 7. SGL Skill
Hermes/OpenClaw skill for integration.
- Already has Hermes skill support
- One-click setup
