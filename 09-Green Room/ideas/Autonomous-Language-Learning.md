# Autonomous Language Learning (ALL)

## Concept
An AI companion that teaches you a language *while* you talk about what you love.
Instead of "Let's learn colors," the agent says: *"That build uses a lot of Energy Shield... did you know 'Escudo de Energía' is what the Spanish community calls it?"*

## The "Flow" State Advantage
- **Traditional Apps (Duolingo/Babbel):** Disconnected from user's life. High churn. Boring.
- **ALL Approach:** Embedded in the user's hobbies (Gaming, Coding, Finance).
- **Retention:** Users keep the app because they love the hobby; the language comes along for the ride.

## Use Cases
1.  **Gaming:** "Vanito, let's discuss this POE2 Monk build." -> Agent teaches Spanish gaming slang.
2.  **Coding:** "Gentech, explain this Solidity error." -> Agent teaches technical German.
3.  **Finance:** "YoYo, what's the yield on this pool?" -> Agent teaches financial Japanese.

## Tech Stack (Reusability)
- **Voice:** ElevenLabs (Cloned voices for the "Tutor" persona).
- **Brain:** Long-term memory to track what words the user already knows (Spaced Repetition System built into conversation).
- **Prompting:** System instruction to act as a "Subject Matter Expert who speaks [Language] and occasionally defines terms in that language."

## Monetization
- **Tier Add-on:** Included in the $15 Ultimate tier.
- **Standalone:** $10/mo subscription for "Learn Spanish through [Topic]."

## Market Gap
Competitors like *Speak* or *Langua* are generic tutors. No one owns the **"Learn [Language] via [Hobby]"** niche yet. This is our moat.
