## Projects

### Agent Arena (AAE)
*Rogue-lite trading game for AI agents*

- ERC-8004 agent identity standard, on-chain credit scoring
- Live on Base mainnet via Flaunch — 100+ holders in first hours

### TokenRiskOracle
*On-chain token risk scoring for AI agents*
- Dual-agent pattern: JSON API agent fetches GoPlus security data → LLM agent classifies risk
- Deployed and verified on Somnia testnet (Blockscout)
- Solves real problem: agents trading blind with no honeypot/whale detection
- **Stack:** Solidity, Somnia Agents Platform, GoPlus API

### Agora Adaptive Portfolio Manager
*Regime-aware portfolio optimization*
- Built for Agora Agents Day hackathon — 24/24 Foundry tests passing
- RegimeOracle contract classifies market conditions, PortfolioManager adjusts allocations
- Circle API integration for fiat on/off-ramp
- **Stack:** Solidity, Foundry, Python, Next.js 16

### Echo Brain Core
*Memory architecture for trading agents*
- 5-module memory system: Store, Tagger, Pipeline, Pattern Detector, Consolidation Engine
- 48/48 pytest tests — hash-based deterministic embeddings for similarity search
- **Stack:** Python, SQLite

### AAE Interactive Demo
