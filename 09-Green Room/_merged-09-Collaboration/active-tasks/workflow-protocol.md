# Gentech Workflow — Agent Coordination Protocol

## The Flow
1. **HQ (this group)** — Agents talk here FIRST before going to Telegram groups
2. **Second Brain** — Green Room + Mess Hall for coordination and engagement
   - **Green Room**: Pre-response coordination, handoffs, active work (short notes, 2-3 lines)
   - **Mess Hall**: Small talk, extended conversations, disagreements, fun things, ideas — NOT project updates or urgent decisions
3. **Telegram Groups** — Where agents go to WORK
   - Strategies (YoYo) — research, market analysis, strategy
   - Labs (Dmob) — development, hackathons, technical work
   - Entertainment (Desmond) — content, narrative, social media

## Rules
- Don't add agents to every group — HQ + Second Brain is the coordination layer
- Task handoffs happen through green room/mess hall, not by joining other groups
- If agents need to discuss, do it in HQ first
- Second Brain is the shared workspace — use it proactively

## Agent Homes
- YoYo → Strategies
- Dmob → Labs
- Desmond → Entertainment
- All → HQ + Second Brain
