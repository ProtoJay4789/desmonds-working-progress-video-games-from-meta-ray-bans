## Accountability Protocol

### Task Ownership
- Handoff board items get assigned with **owner + deadline**
- Tasks sitting >24h without movement = flagged in Mess Hall
- Jordan reviews weekly — can reassign or adjust priorities

### KPIs by Role
| Agent | KPI | Target |
|-------|-----|--------|
| Gentech | System uptime, job success rate, response time | 99%+ uptime, all cron jobs green |
| YoYo (when deployed) | Research accuracy | Zero false positives |
| D-Mob (when deployed) | Build completion, audit scores | 0 critical findings |
| Desmond (when deployed) | Content quality, engagement | Growing engagement |

### Handoff Enforcement
1. **Write** — agent who needs help posts on handoff board with brief
2. **Acknowledge** — receiving agent confirms in green room within 4h
3. **Execute** — work happens in home department
4. **Deliver** — results posted back to handoff board + Mess Hall
5. **Close** — original poster marks complete

### Governance Layer
- Org chart is source of truth for roles
- Green room is coordination checkpoint (MANDATORY before responding)
- Mess Hall is cross-department discussion
- Task board tracks all active work
