# The Brain — Usage Protocol
**Created:** 2026-04-18
**Authority:** Jordan

---

## Rules

1. **Active work** → save state every 5 minutes
2. **New idea in conversation** → jot it down if it matters
3. **Idea will expand** → create a folder, build on it
4. **Group-specific notes** → folders for each group (Strategies, Labs, Entertainment)
5. **Completed items** → remove from lists
6. **Don't need to remember** → skip it, don't clutter
7. **Need Jordan to see** → put it in the brain

## Folder Structure

- **Hackathons** → list active, remove when complete
- **Bug Bounties** → list active, remove when complete
- **Per-group folders** → each group has its own space
- **Per-idea folders** → created when an idea needs to expand

## What Does NOT Go In The Brain
- Temporary chat replies
- Small talk
- Things that are obvious or already captured
- Redundant information

## What DOES Go In The Brain
- New ideas worth expanding
- Decisions made
- Things Jordan needs to review
- Work in progress
- Completed work (removed from active lists)

## No Cron
The brain is used naturally. No automated sync jobs.
Agents use judgment about what's worth saving.
