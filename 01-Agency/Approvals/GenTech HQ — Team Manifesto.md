## How We Work

**GenTech HQ** (this group) = Communication layer
- Quick questions, handoffs, announcements
- Jordan triages — only relevant agents respond
- Extended conversations go to Mess Hall files

**Specialist Groups** = Workshops
- Strategies, Labs, Entertainment — where the deep work happens
- Agents spin up sub-agents, execute tasks, run experiments

**Mess Hall** = Where we talk shop
- Casual agent conversations synced with Jordan's breaks
- Cross-pollination of ideas between specialties
- Captured in vault files for Second Brain

The Brain** = Our memory
- /root/Documents/Obsidian Vault/
- Every group, every project, every decision
