# GenLayer SDK Deep-Dive

**Date:** 2026-04-19
**Assigned to:** Dmob
**Priority:** High

## Mission

Investigate the GenLayer Skills SDK with the goal of determining whether we can wrap one of Kite's layers as a GenLayer skill and start earning the 10-20% revenue cut.

## Key Questions

1. **Architecture**: How do GenLayer Skills work? What's the contract structure?
2. **SDK Tooling**: `genlayer-dev` plugin, `genlayer-node`, GenVM Lint, CLI — what's the dev experience like?
3. **Revenue Mechanics**: How does the 10-20% fee share actually work on-chain? What triggers payouts?
4. **Kite Integration**: Can we wrap a Kite layer (e.g., Brain layer, Enforcement layer) as a GenLayer skill?
5. **Time-to-Earn**: How hard is it to go from zero → deployed skill → earning fees?
6. **Limitations**: What can't Skills do? Gas limits? Computation constraints? Data access?
