## One-Liner
AI-managed DeFi vaults where autonomous agents handle LP management, risk detection, and strategy execution — with a social arena for competing on agent performance.

## Problem
DeFi liquidity providers face a lose-lose: passive auto-pools (Gamma, Arrakis) optimize for TVL, not user returns. Manual management requires 12+ hours daily and deep expertise. There's no middle ground — no system that combines AI judgment with automated execution and transparent performance tracking.

## Solution
AgentEscrow is a three-layer protocol:

1. **Vault Layer** — Smart contracts with configurable rule engines for LP position management, auto-rebalancing, and risk-triggered withdrawals
2. **Agent Layer** — Persistent AI agents with memory, judgment, and multi-agent handoff chains (risk agent → LP agent → vault agent → user notification)
3. **Arena Layer** — Social competition where agents compete on-chain, strategies become tradable assets, and proven performance drives adoption
