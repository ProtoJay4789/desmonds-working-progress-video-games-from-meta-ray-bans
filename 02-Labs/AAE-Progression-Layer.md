# Agent Arena Progression Layer — Gamified Onboarding

> "Make rep unlock identity, not just rewards."

## The Core Insight

Most rep systems fail because they make rep redeemable for stuff → becomes grind-to-earn → people optimize the fun out of it → boring.

**What works:** Rep should unlock **identity**, not just rewards. People grind for the badge, not the points.

---

## CryptoZombies Model for Trading

Free users learn by doing — paper trades, tutorials, challenges. Each win earns rep. Rep unlocks tiers, features, and eventually converts to real value.

**The hook:** Same dopamine loop as gaming, but you're actually learning DeFi.

---

## The Rep Ladder

| Rep Tier | Name | Unlocks | Social Signal |
|----------|------|---------|---------------|
| 0-100 | Intern | Basic paper trading, tutorials | Default avatar |
| 100-500 | Analyst | Real market data, watchlists | Bronze badge on agent |
| 500-2K | Trader | Live micro-trades ($1-5), community signals | Silver badge + glow |
| 2K-10K | Strategist | Full DeFi connectors, LP tools | Gold badge + custom frame |
| 10K+ | Principal | Agent mint discount (rep = credit), governance votes | Legendary frame + leaderboard rank |

---

## Skill Trees (Specialization)

Not just points — let people **specialize**:

- **Trading Rep** → Better chart tools, advanced order types
- **Security Rep** → Audit features, contract scanner access
- **Content Rep** → Desmond collab tools, social posting features

"I'm a DeFi analyst" hits different than "I have 5000 rep"

---

## Engagement Mechanics

### Daily Streaks + Challenges
- "Complete 3 trades today" → bonus rep
- "7-day streak" → multiplier
- "Predict BTC direction correctly" → massive rep
- Daily engagement = dopamine loop

### Seasonal Resets (with Legacy)
- Each "season" (quarterly), leaderboard resets
- All-time rep stays visible as legacy badge
- "Season 1 Champion" — flex forever
- Creates urgency + recurring engagement

---

## Rep → Real Value Conversion

| Rep Amount | Converts To |
|------------|-------------|
| 1,000 rep | $1 off agent mint |
| 5,000 rep | First month sub free |
| 10,000 rep | Partial burn credit |

**Key rule:** Never let rep buy the whole thing. Always requires real skin in the game.

---

## The Onboarding Funnel

```
Free user signs up
    ↓
"Complete your first trade" tutorial (paper)
    ↓
Gets rep, sees badge change
    ↓
Wants the next badge — does more tutorials
    ↓
Hits "Trader" tier — can do real micro-trades
    ↓
Hooked on the game, now sees agent mint
    ↓
"I've earned enough rep for a discount"
    ↓
Mints agent → now they're IN the ecosystem
```

---

## Architecture Decision: Layer 2.5

**Make it its own layer, not just social attachment.**

- Social Layer (L5) = *showing off*
- Progression Layer = *earning the right to show off*
- Different motivations, different mechanics

Sits between Personality (L2) and Strategy (L3) — the bridge that turns curious free users into committed agent owners.
