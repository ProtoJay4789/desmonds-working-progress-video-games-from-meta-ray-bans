**Last Updated:** April 28, 2026
**Source:** Colosseum Copilot API (5,400+ projects) + GitHub

## Executive Summary

Agent payment infrastructure is a **hot space** with 325+ projects on Solana alone. However, **nobody has the full trust stack**: Registry + Escrow + Dispute Resolution + Reputation. Most competitors are "Stripe for agents" — payment routing only.

**Our edge:** We solve trust, not just payments.
