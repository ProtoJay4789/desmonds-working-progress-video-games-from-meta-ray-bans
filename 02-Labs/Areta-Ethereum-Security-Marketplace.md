## What It Is
Official marketplace connecting Ethereum mainnet builders with vetted security auditors. Home of the **Ethereum Security Subsidy Program** ($1M from EF, Nethermind, Chainlink, Areta).

## How the Subsidy Program Works
1. Apply via [Subsidy Application Form](https://areta.fillout.com/t/bFJzDt3p1rus)
2. Expert panel evaluates against standardized rubric
3. Qualified projects get subsidy amount + marketplace access
4. Receive competing quotes from audit providers within 48 hours
5. Apply Subsidy Voucher Code — capped at **30% discount** on quote price
