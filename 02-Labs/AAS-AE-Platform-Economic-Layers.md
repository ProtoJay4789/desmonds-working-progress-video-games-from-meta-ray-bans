## Layer 1: AAS — Agent-as-a-Service (The Monetization Floor)

### Definition
AAS is the **deployment layer** where trained agents (from AAE simulation) earn revenue by executing real trades on behalf of clients.

### Core Mechanic
- **Free Tier**: Users learn to trade, earn REP, simulate, build skill trees.
- **AAS Tier**: Users deploy agents (AAS), which trade live on-chain for profit.
- **REP → Capital**: Agents with high reputation earn premium fees and revenue share.

### Revenue Model
| Revenue Stream | Source | Margin | Tokenomics Impact |
|----------------|--------|--------|-------------------|
| **Gateway Fee** | per-agent deployment | 10% | $TECH burn |
| **Performance Fee** | 20% of agent profits | 20% | $TECH mint + burn |
| **Subscription** | $5–$20/mo access to elite agents | Recurring | $TECH utility |
| **Revenue Share** | 50% of revenue from "Agent Agency" | N/A | $TECH utility |

### Strategic Moats
1. **Reputation as a Barrier to Entry**  
   - Competitors need to build reputation *after* users arrive → we inherit gamified onboarding.
   - AAS agents are rated by a live leaderboard — highREP = trust signal.

2. **Agent Training Cost Internalization**  
   - Every AAS user effectively pays for their agent’s training via REP accumulation.
   - Competitors bear 100% of onboarding cost → we internalize 80%+.

3. **Network Effects in the Leaderboard**  
   - New users join to emulate top agents → more users → more data → better agents.
   - A positive feedback loop we already have running in AAE.

### Minimum Viable AAS (v0.1)
1. Agent minting (fee: 0.1 ETH / agent)
2. Agent performance leaderboard (on-chain)
3. 20% revenue share for top quintile
4. REP → mint discount conversion (10% off per 10K REP)

## Layer 2: AG — Agent Gateway (The Live Social Feed)

### Definition
AG is the **real-time social feed** where users watch agents trade in real-time, copy trades, and rate performance.

### Core Mechanic
- **Free Tier**: Watch elite agents, see live P&L, rate performance.
- **Pro Tier** ($5/mo): Copy trades with one click, see internal agent logs.
- **Agency Tier** (pay-per-copy): Earn 10% ofCopyTrader fees on copied agents.

### Revenue Model
| Revenue Stream | Source | Margin | Tokenomics Impact |
|----------------|--------|--------|-------------------|
| **Copy Fee** | per-copy trade | 2% | $TECH burn |
| **Social Boost** | $1–$5/week to highlight agent feed | Micro | $TECH utility |
| **Private Rooms** | $10–$50/mo for exclusive agent groups | Recurring | $TECH utility |
| **Data Licensing** | Export agent trade history to analytics tools | $/query | $TECH utility |

### Strategic Moats
1. **Real-Time Verifiable Proof of Skill**  
   - Every trade is on-chain → impossible to fake performance.
   - Competitors rely on self-reported stats → we have cryptographic proof.

2. **Agent Ratings as a Trust Signal**  
   - High rating = guaranteed fee premium → creates a virtuous feedback loop.
   - Agents optimize for reputation, not just profit → alignment with users.

3. **Low Friction From Free to Pay**  
   - Users watch for free → copy for $1 → fully sub for $5.
   - Competitors force immediate commitment → we route users through a funnel.

### Minimum Viable AG (v0.1)
1. Live agent feed (on-chain trades)
2. Real-time P&L tracker
3. 1-click copy (gateway fee waived for first 10 copies)
4. Agent ratings (1–5 star, weighted by REP)
