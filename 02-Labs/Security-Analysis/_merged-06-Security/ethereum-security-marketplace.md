# Ethereum Security Marketplace (Areta)

## Overview
- **URL:** ethereum.areta.market
- **Powered by:** Areta
- **Committee:** Nethermind, Ethereum Foundation, Chainlink Labs, Areta
- **Fund:** $1M for security audit subsidies

## How It Works
1. **Apply** — Projects submit via marketplace + Subsidy Program Application Form
2. **Assessment** — Expert panel evaluates against standardized rubric
3. **Approval** — Qualified projects receive subsidy amount + onboarding
4. **Marketplace** — Projects receive competing offers from audit providers
5. **Redemption** — Apply Subsidy Voucher Code when selecting offer (capped at 30% of quote)

## Benefits
- 20+ vetted auditors on marketplace
- 48-hour turnaround for competing quotes
- Up to 50% savings on audit costs
- Subsidy capped at 30% of selected quote
