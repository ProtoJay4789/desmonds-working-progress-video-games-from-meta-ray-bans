## The Vision
> "Do good at simulated trades, earn actual money to put into coins you already want to buy. We make money off the swaps anyway."

## The Loop

```
User plays simulation game (FREE)
    ↓
Demonstrates skill → earns $TECH tokens
    ↓
$TECH redeems for REAL trading capital
    ↓
Capital deploys into tokens user already wanted
    ↓
Gentech earns swap fees on every deployment
    ↓
Good sim trader = real trader with funded account
    ↓
Shares results → brings friends → more sim users → more swaps
```

## Why This Is Different

| Existing | Sim-to-Real |
|----------|-------------|
| Paper trading (no reward) | Earn tokens for skill |
| Funded accounts ($100K eval) | Free simulation, real rewards |
| Learn to earn (boring) | Game to earn (fun) |
| Swap fees go to DEX | Swap fees fund the ecosystem |

## The Prop Firm Model (Reimagined)

Traditional prop firm:
- Pay $500 for evaluation
- Pass → get funded account
- Fail → lose $500

Gentech model:
- Play simulation games (FREE)
- Prove skill over time
- Earn $TECH → redeem for real capital
- Deploy into tokens you want anyway
- Gentech earns swap fees
- **Nobody loses money learning**

