## NFT Mechanics

**How it works on-chain:**
1. Agent registers on `agent_registry` program
2. Character NFT minted via SPL Token (metaplex standard)
3. NFT metadata includes: role, personality, reputation score
4. Visual "wear" updates based on on-chain activity (more jobs = more weathered)

**Trait rarity:**
- Base character: Common (same for all agents in same role)
- Expression variations: Uncommon (5 variations per role)
- Background effects: Rare (performance-based)
- Full weathering: Legendary (100+ completed jobs)

**Revenue model:**
- Mint fee: 0.1 SOL (covers gas + protocol revenue)
- Secondary royalties: 5% back to protocol
- Character trading on Tensor/Magic Eden = ecosystem growth
