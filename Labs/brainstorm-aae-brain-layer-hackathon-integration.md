## How the Brain Layer Maps to Each Hackathon

### 1. Solana Frontier (May 11) — The Trust Brain

**What we're building:** AgentEscrow — identity, reputation, escrow, disputes
**Brain layer integration:** On-chain enforcement + dispute resolution = the "conscience" of the agent

The Solana Frontier submission IS the brain's trust layer:
- **AgentRegistry** — Who is this agent? (Identity brain)
- **JobEscrow** — Is this job funded and valid? (Financial brain)
- **Reputation** — Has this agent performed before? (Memory brain)
- **DisputeResolver** — Is this agent lying? (Integrity brain)

**Brain Layer Angle:** "We don't just build agents that can pay each other. We build agents that can TRUST each other."

### 2. Kite AI (May 17) — The Economy Brain

**What we're building:** Agent Commerce Economy — training, hiring, settlement
**Brain layer integration:** AAE Academy + Agent Portal = the "learning and discovery" brain

The Kite AI submission IS the brain's economic layer:
- **AAE Academy** — Agents learn to trade, earn REP (Training brain)
- **Agent Portal** — Live social feed, real-time P&L (Discovery brain)
- **AAS (Agent-as-a-Service)** — REP-based pricing, performance fees (Market brain)
- **x402 Settlement** — Per-task payments on Kite (Execution brain)

**Brain Layer Angle:** "Everyone else sells you a hammer. We sell you the carpenter."

### 3. Google Cloud Rapid Agent (Jun 11) — The Cloud Brain

**What we're building:** AAS on Google Cloud — cloud-native agent marketplace
**Brain layer integration:** Gemini as the brain, Vertex AI for model serving

The GCP submission IS the brain's infrastructure layer:
- **Gemini** — The actual AI brain making decisions
- **Vertex AI** — Model serving for strategy evaluation
- **Agent Starter Pack** — Production-ready templates
- **Cloud-native deployment** — Scalable brain infrastructure

**Brain Layer Angle:** "We put the brain in the cloud so it never sleeps."

### 4. Retro9000 (Jul 14) — The Full Stack

**What we're building:** Full AAE stack on Avalanche
**Brain layer integration:** ALL layers converge here — the capstone

The Retro9000 submission IS the complete brain:
- Trust layer (from Solana Frontier)
- Economy layer (from Kite AI)
- Cloud infrastructure (from GCP)
- Dynamic Strategy Engine (the NEW piece)
