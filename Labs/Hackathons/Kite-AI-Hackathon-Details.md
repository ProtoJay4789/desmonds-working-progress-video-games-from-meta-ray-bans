## Mission
> Build autonomous AI agents that operate and settle on Kite — the first AI payment blockchain.

Kite AI is purpose-built infrastructure for autonomous agents to transact with identity, payment, governance, and verification. You're building agents that perform real tasks and settle on-chain.

---

## Requirements
1. Shows an AI agent that performs a task and settles on Kite chain
2. **Executes paid actions** (API calls, services, transactions)
3. Works end-to-end in a **live demo in production** (Vercel/AWS)
4. Uses Kite chain for **attestations** (proof, auditability)
5. **Functional UI required:** web app or CLI tool
6. Demo must be publicly accessible or reproducible via README
