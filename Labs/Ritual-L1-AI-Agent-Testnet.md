## How to Participate (Testnet)
1. **Join Discord:** discord.gg/CpYbgtqKz
2. **Get access code:** Navigate to `#ritdrip` channel → run `/get_code` → bot DMs you a personal code
3. **Claim faucet:** faucet.ritualfoundation.org → enter wallet + access code → claim 5 Ritual
4. **Install skill system:**
   ```bash
   git clone github.com/ritual-foundat .claude/skills/ritual-dapp-skills
   ```
5. **Configure:** Replace `YOUR WALLET ADDRESS` with the wallet that claimed tokens
6. **Customize prompt:** Default is *"Build me a private multi-modal ChatGPT on-chain."* — can be replaced with any agent idea
7. **Run:** Executes 5 phases, completes automatically
8. **Monitor:** explorer.ritualfoundation.org
