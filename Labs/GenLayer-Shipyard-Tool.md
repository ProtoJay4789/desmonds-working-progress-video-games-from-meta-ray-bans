**Date:** 2026-04-24
**Source:** https://x.com/i/status/2047684672780427452

## What It Is
- **Shipyard** — No-CLI deployment platform for GenLayer
- Clean interface, no wrestling with setup/config
- Live at: [gen-shipyard.vercel.app](https://gen-shipyard.vercel.app)

## Key Features
- **Network switching** from dashboard:
  - Bradbury
  - Asimov
  - StudioNet
  - Localnet
- Paste code → get live transaction hash in seconds
- Grab contract address → wire into frontend instantly

## Demo App
- **EVMValidator** — live at [evm-validator.vercel.app](https://evm-validator.vercel.app)
- Verifies addresses through GenVM
- Built end-to-end in under 10 seconds with AI coding agent (Antigravity)
