# Obsidian Kanban Plugin
**Date:** 2026-04-24
**Source:** https://github.com/obsidian-community/obsidian-kanban
**Requested by:** Jordan

## What It Is
- Official Obsidian community plugin for Kanban boards
- Creates visual task boards inside Obsidian notes
- Supports drag-and-drop, columns, cards, dates, tags

## Use Case for GenTech
- Visual task tracking inside the vault
- Could integrate with our existing todo/cron workflow
- Alternative to external Kanban tools (Trello, Linear)

## Installation
- Available via Obsidian Community Plugins
- Or manual install from GitHub releases
