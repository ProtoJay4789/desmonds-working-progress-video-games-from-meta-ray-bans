# Jordan's Solidity Origin Story

**Date captured:** April 19, 2026

Been trying to learn Solidity for years. Kept hitting the same wall — understood the basics individually but couldn't "put the code together." Retried school, rewatched Cyph, tried other schools — same result every time.

Then AI tools started coming out. Started with Copilot (code alongside, helpful hints). Then vibe coding emerged. Then Claude Code. That was the unlock. The handicap that held him back for years completely disappeared.

**Majin Buu / Blind Boy Analogy (DBZ):**
Buu — the big bad — finds a blind boy walking on a rock. Tries to scare him but the boy can't see. Can't be scared of what he can't perceive. Buu, obsessed with being menacing, ends up healing the boy's eyes instead. The boy is so grateful he can finally see that he doesn't care what Buu looks like. This interaction starts changing Buu — leads to Hercule friendship, good Buu arc.

**The parallel:** AI was the unlock that gave Jordan "sight" — not just tools, but the ability to finally SEE how to put code together. And in return, Jordan is building something that will change how agents interact (AAE). Mutual transformation.
