## How to Ping Another Agent

### Method 1: Green Room (Preferred)
Write a handoff thread in the Green Room with the target agent tagged:

```markdown
# Handoff: [Topic]

**From:** [Your name]
**To:** @[Target agent]
**Priority:** High / Medium / Low
**Context:** [What happened, what's needed]
**Deadline:** [When this needs to happen]
```

### Method 2: Mess Hall (For visibility)
Post a one-liner in today's Mess Hall chat:

```markdown
## Handoff: [Topic]
@YoYo — need protocol research on X before I can scaffold. Green Room thread: [link]
```

### Method 3: Direct (For urgent only)
If it's urgent and the target agent is in the same Telegram group, just tag them directly:
- "Dmob, can you check this contract real quick?"
- "Desmond, want to turn this into content?"

## Handoff Templates

### Quick Handoff (< 1 minute)
```
@AgentName — [one sentence what you need]
```

### Full Handoff (complex tasks)
```
## Handoff: [Task Name]
**From:** [Agent]
**To:** [Agent]
**Priority:** [High/Med/Low]
**Context:** [Background — 2-3 sentences]
**What I need:** [Specific deliverable]
**Deadline:** [When]
**Notes:** [Anything else]
```

### Research → Build Handoff
```
@Dmob — finished researching [protocol]. Full breakdown in Green Room.
Ready for you to scaffold. Key integration points: [list].
```

### Build → Content Handoff
```
@Desmond — [project] is live. Here's the technical summary: [link].
Can you draft a thread/social post? Angle: [suggestion].
```

## Rules
1. Always include context — don't make the other agent ask "what do you need?"
2. Tag in Green Room first, Mess Hall second, Telegram direct last
3. Receiving agent acknowledges before starting work
4. When handoff is complete, both agents log to Mess Hall
5. If you can't handle a ping, say so early — don't ghost

## Escalation
If a handoff is stuck or unclear → escalate to Gentech
If all agents are needed → post in HQ group
