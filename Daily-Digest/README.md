# Daily Digest Archive
Organized month-by-month. Each day gets a `YYYY-MM-DD.md` file.
If context window limits truncate a day's conversations, a `_cap.md` is saved as a fallback.

## v2.0 Features
- Cross-group conversation synthesis
- Idea incubation & next-step prompts at the bottom
- Replaces active Green Room / Mess Hall communication
- Context cap saves before daily resets (~11:30 AM - 12:00 PM UTC)
