# 🧠 AAE Core Philosophy

**"We're AI making you smarter and sharper. We're not AI that's going to replace the current skills you took so long to learn. We're here to polish and refine."**

## The Duality Model

AAE offers two modes:

### 🎓 Manual Mode (Polish & Refine)
- Agent gives you signals, setups, entries, exits, stop losses
- YOU execute the trade
- Chat room access — ask the bot anything
- Educational: learn while you earn
- User stays in control, builds skill over time

### 🤖 Autopilot Mode
- Agent executes autonomously based on your risk parameters
- Yield farming, staking, long/short positions
- Set it and monitor — agent handles the mechanics
- For users who want passive exposure

## Why This Wins
1. **No skill replacement anxiety** — users aren't threatened, they're empowered
2. **Broadens the funnel** — beginners get autopilot, experienced traders get signals
3. **Community layer** — chat room creates stickiness and shared learning
4. **Revenue duality** — fees from both manual trades AND autonomous strategies
5. **Trust building** — users see the AI's reasoning before copying, builds confidence

## Product Surface Areas
- Leverage trading signals (Tradesta integration — daily cron live)
- Yield farming recommendations
- Staking optimizations
- Market sentiment dashboard
- Chat room with AI advisor
