# Agentic Commerce Protocols Glossary

*Sourced from [@ampersend_ai](https://x.com/ampersend_ai/status/2047425404621991939) — April 23, 2026*

## Active Protocols

| Protocol | Author | Description |
|----------|--------|-------------|
| **x402** | @CoinbaseDev | HTTP-native payment protocol built on the 402 status code. Agents pay for resources inline — no wallets, no human checkouts. |
| **A2A** (agent-to-agent) | @googlecloud | Open protocol for agents to discover, communicate with, and delegate tasks to other agents across different frameworks. |
| **ADK** (agent development kit) | Google | Open-source toolkit for building and orchestrating multi-agent systems. Designed to work natively with A2A. |
| **ERC-8004** | Ethereum | Emerging standard for defining agent spending permissions onchain. Humans set limits on what agents can authorize. |
| **MPP** (machine payment protocol) | — | Protocol for machine-to-machine payments. Focused on low-latency, automated settlement between software agents. |
| **ACP** (agent communication protocol) | @Cisco | Framework for standardizing how agents exchange context, state, and instructions across enterprise environments. |
| **AP2** (agent payment protocol) | — | Payment layer designed specifically for agent workflows. Handles authorization, routing, and settlement between autonomous systems. |
