# Ampersend — Competitive Analysis

**Date:** 2026-04-29
**Source:** @ampersend_ai / ampersend.ai
**Status:** Live product

## What It Is
Agent payment management platform. Create wallets + budgets for AI agents. Agents can get paid and spend money. You control the budgets.

**Traction:** 75K views, 247 likes, 81 reposts — solid signal.

**Distribution:** Available as an OpenClaw skill at ampersend.ai/skill.md

## Comparison: Ampersend vs Kite Passport

| Feature | Kite Passport | Ampersend |
|---------|--------------|-----------|
| Identity | 3-layer (User→Agent→Session) | Wallet per agent |
| Payments | X402 + settlement chain | Budget-based spending |
| Focus | Identity & verification | Payment execution & budgeting |
| Stack | On-chain (Kite chain) | Likely centralized/hybrid |

## Strategic Take
- **Kite Passport** = identity layer ("who is this agent?")
- **Ampersend** = payment execution layer ("how does this agent pay?")

These aren't direct competitors — they solve adjacent problems. Kite verifies identity, Ampersend manages the money flow.

## Relevance to GenTech
- Ampersend is already in the x402 tier list (Active tier)
- OpenClaw skill distribution = developer-first strategy
- Could be a **partner** (plug Ampersend wallets into Kite Passport identity) or **competitor** (if they add identity)
- Our AgentEscrow + Attestation layer is the differentiator: verifiable, not just spending
