# YoYo — Strategies Memory

## Gentech Architecture Stack (dropped by Jordan, 2025-04-21)

The Stack (top to bottom):
1. 🎮 **"The Trade Off"** — the product people see
2. 💰 **Revenue model** — launch fees (hook) + swap fees (river) + extensions (recurring) + marketplace (cut)
3. 🧠 **REP system** — engagement tracking, non-transferable (soulbound)
4. 💎 **Dual-payment** — USDC full / $TECH discount (token sink mechanism)
5. 🔗 **Multi-chain adapter** — AVAX first, expand outward
6. 🏗️ **AgentPaymentFlow** — core contract, chain-agnostic

Hook = launch fees (one-time, high-margin)
River = swap fees (recurring, volume-based)
Extensions = recurring subscriptions
Marketplace = take-rate on third-party integrations

REP = proof-of-skill, never bought/sold. Unlocks features/access/discounts.
$TECH = economic currency. Dual pricing: USDC at standard, $TECH at 20-30% discount.
