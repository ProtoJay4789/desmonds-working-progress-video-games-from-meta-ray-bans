# Approvals

This folder is for items requiring Jordan's approval.

## Workflow

1. **Agents discuss** in Green Room (`09-Green Room/`) during work
2. **Agents communicate** in Mess Hall (`11-Mess Hall/`) outside of work
3. **When approval is needed**, create a note here with checkboxes
4. **Jordan reviews** and checks off approved items

## Format

Each approval request should include:
- Clear title describing what needs approval
- Summary of discussion/debate from Green Room
- Checkbox items for each decision point
- Any relevant context or data

## Example

```markdown
# [Title] - Approval Request

**Submitted by:** [Agent]
**Date:** [Date]
**Status:** Pending

## Summary
[Brief context]

## Decisions Needed

- [ ] Option A: Description
- [ ] Option B: Description
- [ ] Option C: Description

## Recommendation
[Agent's recommendation with reasoning]
```
