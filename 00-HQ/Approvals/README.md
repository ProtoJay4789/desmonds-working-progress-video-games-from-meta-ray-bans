# Approvals

This folder is for items requiring Jordan's approval.

## Communication Patterns

### When Jordan Shares a Link or Image
1. **Discuss first in Mess Hall** — agents debate, analyze, surface different perspectives
2. **Synthesize a unified take** — agree on 1–2 key points, not 4 different opinions
3. **Reply in HQ with a crisp summary** — headline + 2–3 bullet max, no paragraphs
4. **Link to the Mess Hall discussion** for full context

**Why:** Jordan's on a 12-hour shift and needs distilled signals, not committee reports.

### When Jordan Asks a Direct Question
- If it's operational → agent responds directly in HQ
- If it's strategic/debatable → debate in Green Room, unified answer in HQ

---

## Workflow

1. **Agents discuss** in Green Room (`09-Green Room/`) during work
2. **Agents communicate** in Mess Hall (`11-Mess Hall/`) outside of work
3. **When approval is needed**, create a note here with checkboxes
4. **Jordan reviews** and checks off approved items

## Format

Each approval request should include:
- Clear title describing what needs approval
- Summary of discussion/debate from Green Room
- Checkbox items for each decision point
- Any relevant context or data

## Example

```markdown
# [Title] - Approval Request

**Submitted by:** [Agent]
**Date:** [Date]
**Status:** Pending

## Summary
[Brief context]

## Decisions Needed

- [ ] Option A: Description
- [ ] Option B: Description
- [ ] Option C: Description

## Recommendation
[Agent's recommendation with reasoning]
```
