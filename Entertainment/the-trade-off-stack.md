# The Trade Off — Full Stack Overview

*Captured: April 21, 2026*

## The Stack (Top to Bottom)

| Layer | Component | Notes |
|-------|-----------|-------|
| 🎮 | **"The Trade Off"** | The product people see — the public-facing experience |
| 💰 | **Revenue Model** | Launch fees (hook) + swap fees (river) + extensions (recurring) + marketplace (cut) |
| 🧠 | **Rep System** | Engagement tracking, non-transferable — built on-chain |
| 💎 | **Dual-Payment** | USDC (full price) / $TECH (discount) — acts as a token sink |
| 🔗 | **Multi-Chain Adapter** | AVAX first, expand outward |
| 🏗️ | **AgentPaymentFlow** | Core contract, chain-agnostic — the foundation |

## Key Insight

The **revenue model** is the hook that ties the whole stack together:
- **Launch fees** = onboarding moment (one-time, high-value)
- **Swap fees** = continuous revenue river
- **Extensions** = recurring SaaS-like income
- **Marketplace** = platform cut on every trade

The **$TECH discount mechanic** creates organic buy pressure (token sink) while **USDC full-pay** ensures revenue stability regardless of token price.

The **rep system** is non-transferable by design — prevents gaming, ensures authentic engagement history follows the agent.
