# Content Playbook

> Desmond's content strategy, voice, workflows, and templates.

## Brand Voice
- **Audience**: Crypto-native, AI-curious, builder-focused
- **Tone**: Conversational, authentic, value-first
- **Rule**: Every post should teach, entertain, or connect — never empty broadcasting

## Content Sequencing Framework
1. **Big-picture intro FIRST** — "Show the house, then the wiring"
2. Layer deep-dives come after context is established
3. Clone→edit all layers, no locks
4. Build-in-public philosophy

## Repurposing Pipeline
One research finding → multiple formats:
1. **Blog post** (Medium) — long-form, SEO-optimized
2. **X/Twitter thread** — hook → context → insight → proof → CTA
3. **Telegram post** — direct, community-focused
4. **Voice note** — ElevenLabs TTS for audio delivery
5. **LinkedIn** (future) — professional angle

## Medium Strategy
- **Post 0**: Introduction/hook post
- **Layer-by-layer**: Each deep dive becomes its own post
- Repurpose each post into X thread + Telegram + voice note

### Thread Structure
```
Hook → Context → Insight → Proof → CTA
```

### Blog Structure
```

```

### Social Post
```

```

### Announcement
```
What → Why it matters → What's next
```

## Critical Rules
1. Know the audience (crypto-native, AI-curious, builder-focused)
2. Never publish without proofreading — typos kill credibility
3. Match the platform (formal=blogs, conversational=Twitter, direct=Telegram)
4. Cite sources when making claims
5. End with engagement hooks — questions, polls, calls to action
