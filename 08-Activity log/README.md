# 08-Activity Log — Organized Activity Tracking

## Structure
```
08-Activity log/
├── 2026-01-January/
│   ├── W1/ (Week 1)
│   ├── W2/ (Week 2)
│   ├── W3/ (Week 3)
│   └── W4/ (Week 4)
├── 2026-02-February/
│   └── W1-W4/
├── ... (12 months total)
└── 2026-12-December/
    └── W1-W4/
```

## How It Works
- **Daily notes** go in the current month/week folder
- **Weekly summaries** stay in the week folder
- **Monthly summaries** stay at root level
- **Content drafts** go in month folder's `content-drafts/`
- **Reviews** go in month folder's `Reviews/`
