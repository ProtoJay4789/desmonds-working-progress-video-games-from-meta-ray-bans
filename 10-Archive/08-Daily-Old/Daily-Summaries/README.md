# 📅 Daily Summaries

Short-form daily rollups — one file per day covering agent activity, key decisions, and blockers.

## Retention Policy
- **Keep:** 7 days
- **Auto-purge:** Files older than 7 days deleted during Sunday vault maintenance (`4835b4241e9d`)
- **No weekly/monthly rollups** — just daily, just 7 days, keep it simple

## Format
- `YYYY-MM-DD-summary.md` — Generated by vault maintenance cron or end-of-day wrap-up
