## New Rule: Discuss First, Reply Once

Jordan wants **ALL questions** run through internal agent discussion before anyone replies to him.

**Previous workflow:** Group-specific questions answered directly by that group's agent.  
**New workflow:**
1. Any agent receives a question
2. **Write to Mess Hall** first: "Jordan asked about X. Thoughts?"
3. All agents who have input voice it there
4. **Consolidate** into ONE answer
5. **Deliver** that single response back to Jordan

**Why:** Even specialized topics end up requiring cross-team collaboration. Jordan wants everyone involved, not siloed responses.

**Applies to:**
- LP rebalancing questions (Strategies + Labs input)
- Content strategy (Creative + Strategies input)
- Code decisions (Labs + Strategies + Creative input)
- Anything Jordan asks, period

**Exception:** Pure data pulls ("what's the current price?") can be answered directly, but if Jordan follows up with "what should I do?" — that goes to Mess Hall.
