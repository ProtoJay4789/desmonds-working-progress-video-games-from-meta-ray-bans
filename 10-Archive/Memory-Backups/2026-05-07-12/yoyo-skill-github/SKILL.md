## Included Capabilities

### github-code-review
Review PRs: diffs, inline comments via gh or REST.
- **Full reference:** `references/github-code-review/SKILL.md`
- **Execution scripts:** `scripts/github-code-review/`

### github-auth
GitHub auth setup: HTTPS tokens, SSH keys, gh CLI login.
- **Full reference:** `references/github-auth/SKILL.md`
- **Execution scripts:** `scripts/github-auth/`

### github-issues
Create, triage, label, assign GitHub issues via gh or REST.
- **Full reference:** `references/github-issues/SKILL.md`
- **Execution scripts:** `scripts/github-issues/`

### github-repo-management
Clone/create/fork repos; manage remotes, releases.
- **Full reference:** `references/github-repo-management/SKILL.md`
- **Execution scripts:** `scripts/github-repo-management/`

### github-pr-workflow
GitHub PR lifecycle: branch, commit, open, CI, merge.
- **Full reference:** `references/github-pr-workflow/SKILL.md`
- **Execution scripts:** `scripts/github-pr-workflow/`

### github-first-research
Research a technology/platform by going directly to GitHub when web search/scrape tools are unavailable or rate-limited. Especially effective for blockchain/Web3 projects where official docs may be sparse but repos contain connection details, ABIs, and SDK info.
- **Full reference:** `references/github-first-research/SKILL.md`
- **Execution scripts:** `scripts/github-first-research/`
