# Session Notes: Multi-Application Batching via college.xyz
**Date:** 2026-05-04  
**Platform:** college.xyz/careers  
**Opportunities:** Hibachi Growth Intern, Injective Product Manager Intern, No Limit Holdings Investment Analyst Intern  
**Context:** Three distinct roles requiring different resume variants and cover letter angles, submitted in a single batch after coordinated preparation.

## Workflow
1. **Discovery:** Check existing application drafts in vault (`04-Entertainment/Applications/`) to avoid duplication.
2. **Gap Analysis:** Identify missing cover letters (No Limit Holdings was absent).
3. **Creation:** Draft standalone cover letter for missing role using `career-platform-research` template structure.
4. **Consolidation:** Patch the master application document (`College.xyz-Jordan-Hibachi-Injective.md`) to include:
   - Role overview for new opportunity
   - Distinguishing factors in strategy section
   - Cover letter template
   - Tracking checklist updates
   - Social media templates updated
   - Decision framework expanded to three options
5. **Verification:** Ensure resume variants align with role types (Growth → Growth_Resume, Analyst → Analyst_Resume, Product → Master or tailored).
6. **Delegation:** Message GenTech HQ to notify DMOB of resume verification needs and tracking setup.
7. **Execution:** Jordan submits all applications in one batch the following morning.

## Templates & Artifacts
- **Decision Framework Table:** Compare comp, location, learning, exit ops across multiple offers.
- **Tracking Checklist:** Date submitted, resume used, follow-up date, interview status.
- **Follow-up Protocol:** DM hiring manager on Twitter/LinkedIn after 3–5 business days if no response.

## Key Insights
- Batching applications reduces context-switch and ensures consistent narrative across opportunities.
- Standalone cover letter files improve version control before merging into master doc.
- Always verify resume variant matches role emphasis; tailor further if needed.
- Social signaling (Twitter/LinkedIn posts) can be done pre- or post-submission for awareness.
- Set a 3–5 day follow-up cadence to demonstrate interest without being pushy.
- Use the Green Room (or GenTech HQ) to consolidate before presenting final decision to Jordan.
