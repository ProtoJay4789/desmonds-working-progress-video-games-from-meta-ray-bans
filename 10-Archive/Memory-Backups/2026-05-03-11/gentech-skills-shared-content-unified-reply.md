# Unified Messaging Template — Shared Content Discussion Protocol
# Usage: Copy, fill, and send as single HQ reply after Mess Hall debate

```
🧠 Link Analysis: [Topic/Repo/Paper Name]

Headline: [One-sentence takeaway — keep it sharp]

Key points:
• [Point 1 — ≤10 words]
• [Point 2 — ≤10 words]
• [Point 3 — optional, ≤10 words]

Questions for Jordan:
• [DMOB: Does this need audit/scoping?]
• [YoYo: Finance angle — trackable?]
• [Desmond: Content potential?]

Action: [What we're doing / No action needed]

Full discussion: `11-Mess Hall/YYYY/WXX/YYYY-MM-DD/shared-content-xyz.md`
```

## Instructions

1. **After Step 2 (Mess Hall debate)**, copy this template into your compose window
2. **Fill headline** — one sentence, no punctuation fluff
3. **Key points** — distill the Mess Hall debate to 2–3 bullets, each ≤10 words
4. **Questions** — include every agent's skill-based question (replace brackets with live item)
5. **Action** — give Jordan a clear "we're doing X" or "nothing needed"
6. **Link to Mess Hall** — paste the file path where full debate lives
7. **Send** — this is the ONLY reply in HQ for this link; all other discussion stays in Mess Hall

🧠 Link Analysis: Kite AI Passport Technical Deep Dive

Full discussion: `11-Mess Hall/2026/W18/2026-04-29/shared-content-kite-passport.md`

## Pitfalls to Avoid

- ❌ Don't include your full analysis — keep bullets tight
- ❌ Don't summarize what Mess Hall already debated; distill only
- ❌ Don't omit skill-based questions — each department must contribute
- ❌ Don't ask "Thoughts?" — action-oriented close only
- ❌ Don't forget the file link — Jordan needs to audit the debate
