## Field Explanations

| Field | Value | Source |
|-------|-------|--------|
| `Pool` | Protocol name + contract address | `lp-position-reader.py` decoded |
| `Strategy` | curve / curve-x / spot; fee tier; binStep | Config file `.lfj-aae-config.json` |
| `Price` | Current quoted price in base token | `d5-master-cron.py` (CMC/DexScreener) |
| `IL` | Impermanent loss percentage | `lp-position-reader.py` (on-chain) |
| `Range` | Active liquidity range boundaries | `lp-position-reader.py` (active bin ± range width) |
| `Efficiency` | (Fees earned / capital deployed) × 100 | `lp-position-reader.py` (precise) |
| `Balances` | Token0 amount ($value) + token1 amount ($value) = total | On-chain decoded balances |
| `Action Items` | Checklist for reviewer (YoYo) | Standard process |

## Threshold Configuration

Adjust thresholds in `00-HQ/config/defi-lp-config.env`:

```bash
# IL spike thresholds (percentage)
IL_THRESHOLD_STABLE=2       # For stable pairs (USDC/USDT, stablecoin-stablecoin)
IL_THRESHOLD_CORRELATED=5   # For correlated assets (stETH/ETH, wBTC/BTC)
IL_THRESHOLD_VOLATILE=10    # For uncorrelated volatile pairs

# Efficiency zones
EFFICIENCY_URGENT=30        # ≤30% → immediate rebalance review
EFFICIENCY_WATCH=50         # 30–50% → monitor + DCA boost
EFFICIENCY_HEALTHY=70       # 50–70% → normal monitoring
EFFICIENCY_SILENT=100       # >70% → no action needed
```

## When to Use This Template

✅ **Use for:**
- IL exceeding thresholds (any magnitude)
- On-chain verification needed before rebalance
- Vault entry requires explicit review flag and action checklist

❌ **Do NOT use for:**
- Routine daily updates (use normal vault entry template)
- Price still inside range with IL < threshold (no flag needed)
- Balance-only updates without IL concerns

## Automation Hooks

**Cron pre-check:** Before writing vault entry, `lp-position-reader.py` should assert:
```python
if abs(il_pct) > config['il_threshold_volatile']:
    raise RuntimeError(f"IL spike {il_pct:.2f}% — manual review required")
```

Or, better, **auto-flag but do not abort:** set `action="🚨 Review"` in vault entry and send Telegram alert to `@GenTechStrategies`.
