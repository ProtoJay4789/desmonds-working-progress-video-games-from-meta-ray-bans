# Barrier Lexicon — Phrases That Exclude Beginners

## Financial Barriers
| Phrase | Why It's a Barrier | Action |
|--------|--------------------|--------|
| "stake a small amount of ETH" | Requires crypto wallet + ETH holdings | FILTER OUT |
| "refundable deposit" | Still requires upfront capital | FILTER OUT |
| "entry fee" | Direct monetary cost | FILTER OUT |
| "pay to enter" | Commercial event, not open | FILTER OUT |

## Ambiguous Phrases (Investigate Further)
These require clicking through to the actual requirements page:

| Phrase | Follow-up Question | Possible Outcomes |
|--------|--------------------|-------------------|
| "wallet required" | Is this for prize distribution only, or staking? | If staking → barrier; if prize claim → OK |
| "GitHub account" | Just for submission, or portfolio review? | Submission-only → OK; portfolio review → barrier |
| "student ID" | Only students, or students get bonus? | Student-only → FILTER if user not student; bonus → OK |
| "recommended experience" | Required or suggested? | If "required" → barrier; "suggested" → OK |
| "preferred" | Optional or preferred scoring? | Preferred (not required) → still creates de facto barrier → FILTER OUT |
