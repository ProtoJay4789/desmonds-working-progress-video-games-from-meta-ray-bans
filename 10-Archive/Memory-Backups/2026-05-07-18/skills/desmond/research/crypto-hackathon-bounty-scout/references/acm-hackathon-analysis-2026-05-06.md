# ACM Hackathon Analysis (Swarms Marketplace)

**Source:** https://docs.swarms.ai/docs/marketplace/acm-hackathon
**Date Analyzed:** May 6, 2026

## Hackathon Details

| Field | Value |
|-------|-------|
| **Name** | ACM Hackathon |
| **Platform** | Swarms Marketplace |
| **Prize Pool** | $30,000 |
| **Duration** | May 6 – May 27, 2026 (21 days) |
| **Requirements** | Tokenized agents/prompts via Frenzy Mode |
| **Rewards** | Solana, $SWARMS, USDC |

## What They Want
- Useful, market-ready agents and prompts
- Solve practical problems
- Easy for buyers to understand
- Clear value in real-world workflows

## Gentech Fit Assessment

### High-Fit Submissions (Low Risk)
1. **DeFi LP Monitor Agent** — Position tracking, IL alerts, DCA automation
2. **Multi-Agent Voice System** — ElevenLabs integration, daily briefings
3. **Hackathon Submission Writer** — Technical docs, pitch decks, READMEs

### Strategic Value
- Tokenized agents earn creator fees (revenue stream)
- Solana + Swarms cross-promotion
- Market-ready agent showcase

## Analysis Pattern Used

1. **Green Room coordination first** — Posted handoff before analyzing
2. **Source verification** — Browsed documentation page
3. **Fit assessment** — Evaluated against existing Gentech projects
4. **Risk evaluation** — Categorized submissions by complexity
5. **Strategic recommendation** — Tokenize existing agents, not build new

## Key Insight
This hackathon rewards **tokenizing existing work**, not building from scratch. Ideal for Gentech because we have production-ready agents that solve real problems.
