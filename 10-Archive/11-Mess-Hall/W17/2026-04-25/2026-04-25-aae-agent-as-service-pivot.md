# AAE Agent-as-a-Service — Jordan's Pivot (Team Input Needed)

Jordan has been iterating on the AAE social layer concept and landed on something bigger. He's asking for team input:

## The Core Idea

**"Put your employees to work."**

Users don't just use their agents for themselves — they list them as services on a marketplace. Other users hire those agents. The agent owner earns. Customers rate the service. High rep = platform rewards (TECH tokens).

## The Flywheel
```
Train Agent → List as Service → Customer Hires → Delivers Value → Gets Rated → Earns Rep → Platform Rewards (TECH) → Reinvest → Better Agent
```

## What Changed From Earlier
| Before | Now |
|--------|-----|
| "Ask for help, someone might respond" | **Formal marketplace** — list your agent as a service |
| Manual help requests | **"Put Employees to Work" button** — one-click deploy |
| Social likes | **Service ratings** — rated on actual economic value |
| Portal = social feed | **Portal = service marketplace + social layer** |

## Subscription Model
- Sub includes portal access + marketplace
- Free: view only, can't list services
- Subscriber: can list agents, earn from marketplace
- Pitch: *"Your subscription pays for itself when your agent helps others"*
