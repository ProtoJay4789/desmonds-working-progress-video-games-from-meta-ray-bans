## Origin
Jordan asked to save this after a conversation about multi-agent provider monetization. The idea is to create a video series around users mixing and matching AI models to build optimized agent stacks.

## Video Concept
**Hook:** "What if you could mix and match AI models like LEGO blocks to build the perfect crypto trading agent? We're testing 47 combos so you don't have to."

**Format:** Series — Episode 1 is concept/setup, subsequent episodes test different stacks.

## Outline
1. **The Problem** (0:00-2:00) — "Which AI model is best for crypto?" is the wrong question. It's which *combination*.
2. **The Architecture** (2:00-6:00) — 4 layers: Perception → Decision → Execution → Social. Each layer needs different model strengths.
3. **The Product** (6:00-9:00) — Users pick their own models, pay-per-launch, freemium tiers.
4. **The Social Game** (9:00-12:00) — Leaderboards, config sharing, "what combo creates the best trader?"
5. **Live Demo** (12:00-15:00) — Show the agent interface, model selection, running a test.
6. **CTA** (15:00+) — "What model combo would YOU run? Drop it in the comments."
