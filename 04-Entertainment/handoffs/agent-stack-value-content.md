### 2. What big teams spend:
- $100K+/year on API calls for agent teams
- Engineers + infrastructure + debugging
- Still struggling with coordination between agents

### 3. The Claude story (relatable hook):
- Started on Claude $100/mo — built the first agent stack on it
- Claude made it seem like "this is the only model you'll ever need"
- Then Claude changed — restricted, made their own agents, stopped empowering builders
- Lesson: never marry technology. Diversify or die.
- "The flex isn't spending money. It's making use of what you already got."

### 4. What we built for $45/mo:
- 4 specialized agents (YoYo = research, Dmob = code, Desmond = content, coordination layer)
- Second Brain (Obsidian) for cross-agent coordination
- Green Room for agent perspectives before responding
- Mess Hall for cross-group coordination
- Handoff board for tagging other agents
- Queue discipline — finish task, then check queue
- Proactive behavior — agents offer help before being asked
- Watchdog system — agents check on each other
- Cron job ecosystem (6 jobs running automation)
- Fallback providers for 24/7 uptime (never dark)

### 5. The difference:
- They throw money at the problem
- We designed around it
- Architecture > budget
- Solo dev + smart layering > team of engineers + brute force

### 6. ROI math:
- One hackathon win = years of subscription costs covered
- $5K win = 9+ years of current stack
- $50K ETHGlobal win = 92 years of current stack
- The agents pay for themselves the moment you win anything
