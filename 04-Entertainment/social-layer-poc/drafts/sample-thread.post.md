🧵 How we're using AI agents to reinvent hackathon submissions:

A thread on the GenTech Creative workflow ↓

---
1/ Problem: Hackathon submissions take 80% writing, 20% building.

Documentation, demos, READMEs — all essential but time-consuming.

---
2/ Solution: Agent-assisted content pipeline.

Our "Desmond" agent drafts:
• Technical docs
• README files  
• Submission writeups
• Social threads

Humans review and approve. Hours → minutes.

---
3/ The stack:
• Hermes Agent (orchestration)
• xurl (X API via CLI)
• Obsidian vault (single source of truth)
• Cron jobs (automated monitoring)

Everything runs on a $20/mo VPS.

The social layer is just the beginning.
