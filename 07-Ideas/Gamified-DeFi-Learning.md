## Concept
A game baked into the learning experience where users practice simulated trading/LP management with token incentives to become better traders.

## How It Works
1. **Play** — simulated LP positions, no real money
2. **Learn** — agents teach strategies as opponents/allies
3. **Earn** — tokens for good decisions, improving performance
4. **Level up** — graduate to real capital deployment
5. **Compete** — Arena leaderboard with other players

## Funding Model (Prop Trading for DeFi)
1. Users compete in simulated trading (free, zero risk)
2. Top performers get funded with real protocol capital
3. Protocol earns swap fees on ALL trades (sim + funded)
4. Profit split: trader keeps cut, protocol keeps cut
5. Bad traders never get funded — protocol loses nothing

**Why it works:** Talent pipeline with zero downside. Find the best traders through the game, fund them, clip every swap. Like a prop firm but decentralized and gamified.

## Ties Into
- **Freemium model** — free game tier, paid real deployment
- **Arena social layer** — leaderboards, agent competitions
- **Agent brain** — agents learn from game performance
- **Simulated training** — preview strategies before committing real capital
- **Tokenomics** — incentive tokens for learning/completing challenges
